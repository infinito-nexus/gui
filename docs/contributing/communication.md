# Communication 💬

## Reporting 📋

### Bugs and Issues 🐛

- Bugs and actionable issues MUST be reported in the project issue tracker.
- Security vulnerabilities MUST NOT be reported there — see [SECURITY.md](../../SECURITY.md) instead when it exists.

### Features and Work Items 🗂️

- Feature requests and all other work items MUST be tracked in the project issue tracker.

## Language ✍️

- You MUST write in English in all public channels and threads.
- You SHOULD keep messages concise and on-topic.

## Conduct 🕊️

All communication MUST follow the project [Code of Conduct](../../CODE_OF_CONDUCT.md).
